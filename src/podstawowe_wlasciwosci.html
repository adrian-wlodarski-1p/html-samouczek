Podstawowe właściwości

W CSS jest kilka różnych właściwości, które stanowią podstawę o decydowaniu o wyglądzie elementów.

background-color

background-color, w skrócie background, pozwala zmienić kolor tła elementu.

p { background: yellow; }
<p>Taki sam efekt jak &lt;mark&gt;.</p>

Taki sam efekt jak <mark>.

Od teraz będę używać znacznika <div> do demonstracji właściwości.

width i height

Są to dwie najbardziej podstawowe właściwości znaczników blokowych. Ich działanie jest proste - ustawiają szerokość i wysokość elementu. Najczęściej mierzy się je w pikselach:

div {
  width: 100px;
  height: 100px;
  background-color: blue;
}
<div></div>
		

		
Sczegóły mierzenia wielkości w innych jednostkach będą przedstawione na innej lekcji.

		
Jest także kilka interesujących wartości nie-numerycznych. Pierwszą z nich jest auto, które sprawia, że wielkość bloku sama dostosowuje się do okoliczności:

		
div {
  width: auto; /* domyślne */
  /* height także domyślnie wynosi auto */
  background: blue;
}
<div>Wielkość tego bloku dopasowuje się do tekstu. Jest to zachowanie domyślne.</div>
