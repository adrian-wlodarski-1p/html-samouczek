Pozycjonowanie

Teraz już, gdy przerobiliśmy tak wiele rzeczy, jest czas, by nauczyć się w pełni kontrolować pozycję elementów i zajmowaną przez nie przestrzeń.

Wygląd i przestrzeń elementu

width i height

Są to dwie najbardziej podstawowe właściwości znaczników blokowych. Ich działanie jest proste - ustawiają szerokość i wysokość elementu. Najczęściej mierzy się je w pikselach:

div {
  width: 100px;
  height: 100px;
  background-color: blue;
}
<div></div>

Choć dozwolone są też procenty, mierzone względem elementu nadrzędnego:

#parent {
  width: 100px;
  height: 100px;
  background-color: blue;
}
#child {
  width: 50%;
  height: 50%;
  background-color: red;
}
<div id="parent">
  <div id="child"></div>
</div>

Szczegóły mierzenia wielkości w innych jednostkach są przedstawione gdzie indziej.

Jest także kilka interesujących wartości nie-numerycznych. Pierwszą z nich jest auto, które sprawia, że wielkość bloku sama dostosowuje się do okoliczności:

div {
  width: auto; /* domyślne */
  /* height także domyślnie wynosi auto */
  background-color: cyan;
}
<div>Wielkość tego bloku dopasowuje się do tekstu. Jest to zachowanie domyślne.</div>

Zauważmy, że blok zajmuje domyślnie całą dostępną szerokość. By temu zapobiec, można użyć display: inline-block.

width i height mogą przyjmować wartość inherit, w przypadku których jest ona jednak bezużyteczna.

Kontrola automatycznych wielkości

Okej, ale co jeśli chcemy, by szerokość lub wysokość zmieniała się w razie potrzeb, ale nie przekraczała pewnych wartości? Jeśli width lub height jest ustawione na auto, to możemy regulować ich wartość za pomocą czterech właściwości:

  • min-width określa najmniejszą możliwą szerokość elementu.
  • max-width określa największą możliwą szerokość elementu.
  • min-height określa najmniejszą możliwą wysokość elementu.
  • max-width określa największą możliwą wysokość elementu.

margin i padding

Są to dwie bardzo ważne właściwości.

margin określa, ile jest miejsca wokół elementu:

div {
  margin: 10px;
  background-color: cyan;
}
<div>Ten blok lubi mieć trochę miejsca wokół siebie.</div>

padding określa, ile miejsca jest wewnątrz elementu, ale wokół zawartości:

div {
  padding: 10px;
  background-color: cyan;
}
<div>Ten blok jest bardzo przy kości, ale nie mów mu o tym.</div>

border

border jest to właściwość, która pozwala wyświetlać obramowanie dookoła dowolnego elementu. Przyjmuje ona aż trzy wartości: szerokość obramowania (z jednostką, border-width), typ obramowania (border-style) i kolor (border-color):

border: 3px solid black;

Równoznaczne:

border-width: 3px;
border-style: solid;
border-color: black;
  • border-width pozwala na ustawienie szerokości obramowania.
  • border-style pozwala na zmianę stylu obramowania. Jest kilka wartości możliwości:
    • none/hidden - brak
    • solid - jednolite
    • dotted - kropkowane
    • dashed - kreskowane
    • double - podwójne
  • border-color określa kolor obramowania (słownie lub kod hex).

box-sizing: border-box

Czasem podczas kodowania możesz się natknąć na jeden problem: padding i border zwiększają wyświetlaną szerokość i wysokość, więc width i height mogą nie odzwierciedlać prawdziwej wielkości elementu:

div {
  width: 100px;
  height: 100px;
}
#one { padding: 0; background: red; }
#two { padding: 25px; background: blue; }
<div id="one"><div id="two">

Lecz ustawiając właściwość box-sizing na border-box, można temu zapobiec. Zmienia to sposób obliczania rozmiarów elementu, tak że padding i border są już w nich uwzględnione:

div {
  box-sizing: border-box;
  width: 100px;
  height: 100px;
}
#one { padding: 0; background: red; }s
#two { padding: 25px; background: blue; }

Właściwości kierunkowe

Można ustawić, by margin, padding lub border wpływały na tylko jedną stronę elementu, poprzed dodanie -left, -right, -top lub -bottom na końcu, ale przed przyrostkiem. Na przykład margin stosowane tylko po lewej to margin-left, ale zmiana koloru borderu tylko po prawej to już border-right-color.

Pozycjonowanie

Tak właściwie, to pozycjonowanie to sztuka ustawiania pozycji elementu. Tego się właśnie teraz nauczymy.

Właściwość position

By móc korzystać z pozycjonowania, musisz skorzystać z właściwości position. Jest pięć głównych wartości:

  • static jest domyślne. W takim przypadku pozycjonowanie jest niedostępne.
  • Element z relative zachowuje się normalnie, z wyjątkiem tego, że można go przesunąć za pomocą właściwości pozycji. Oryginalna pozycja elementu wpływa na inne elementy.
  • absolute kompletnie wyłącza element z flowu, tak że on nie wpływa na inne elementy. Domyślnie jest on ustawiony tam, gdzie zwykle, ale dodanie właściwości pozycji sprawia, że przyjmuje on pozycję względem strony, niezależnie od tego gdzie był wcześniej.
  • fixed wyłącza element z flowu i uzależnia jego pozycję od okna przeglądarki. Najdziwniejsze jest to, że taki element nie zmienia pozycji, gdy strona jest przewijana. Domyślnie jego pozycja względem okna przeglądarki jest taka sama, jak początkowo obliczona na podstawie pozycji w dokumencie, ale dodanie właściwości pozycji sprawia, że przyjmuje on określoną pozycję względem okna przeglądarki, niezależnie od tego, gdzie poprzednio był.
  • sticky jest chyba najdziwniejszym pozycjoowaniem. Dopóki element jest w całości wyświetlony w oknie przeglądarki, zachowuje się on jak relative (ale nie przesunięte), lecz w chwili, gdy tylko zbliży się do granicy okna przeglądarki na określoną odległość, to dopóki normalnie byłby za tą granicą, dopóty zachowuje się jak fixed. Właściwości pozycji określają, jak blisko których brzegów przeglądarki element może się zbliżyć.

Właściwości pozycji

Są cztery właściwości pozycji. Ich znaczenie jest zależne od position.

  • relative - Przesuwają element w określonym kierunku.
  • absolute - Ustawiają element względem określonego brzegu strony.
  • fixed - Ustawiają element względem określonego brzegu okna przeglądarki.
  • sticky - Określają, w jakiej odległości od określonego brzegu okna przeglądarki element może się znaleźć.

top oznacza górę, bottom - dół, left - lewą, a right - prawą. Wartości ujemne są dozwolone.

Pozycjonowanie względem elementu

Przedstawiliśmy już cztery rodzaje pozycjonowania: względem samego siebie, względem strony, względem okna przeglądarki i mieszane. Istnieje jednak też możliwość pozycjonowania elementu względem innego elementu. By to osiągnąć, podejmij następujące kroki:

  • Ustaw relative na elemencie nadrzędnym,
  • ustaw absolute na pozycjonowanym elemencie.
