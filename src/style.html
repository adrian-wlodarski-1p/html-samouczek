Stylowanie w CSS

Na poprzedniej lekcji nauczyliśmy się, czym jest HTML i czym są znaczniki. Jednak, technicznie rzecz mówiąc, HTML służy tylko do tworzenia struktury strony. Do modyfikowania jej wyglądu służy inny język, CSS.

Co to CSS?

CSS to specjalny język służący do definiowania stylów, czyli określający sposób, w który napisana w HTML strona się wyświetla. Dokument CSS definiujący style to arkusz stylów. Każda strona ma style, nawet jeśli są one po prostu domyślnymi stylami przeglądarki. Te ostatnie nie są jednak zalecane, bo strona może wtedy wyglądać inaczej na każdej przeglądarce.

Podstawy

Arkusz stylów składa się z tzw. reguł. Każda reguła składa się z selektora i par właściwość-wartość, czyli deklaracji:

selektor { właściwość: wartość; }

Część w klamrach to definicja. Przeanalizujmy więc przykładową regułę:

p { color: blue; }

Jej selektor to p, co oznacza, że ta reguła będzie zaaplikowana wszystkim znacznikom <p>, lecz tylko im. W definicji mamy tylko jedną deklarację. color: blue oznacza, że wszystkie elementy objęte danym selektorym mają niebieski kolor tekstu. Więc, ta reguła zmienia kolor wszystkich znaczników <p> na niebieski.

Kaskadowość

Jest kilka sposobów na dodanie (arkusza) stylów do dokumentu, i są one ułożone w ścisłej hierarchii ważności określającej, które style się bardziej liczą, zwanej hierarchią stylów (od najważniejszego do mniej ważnego):
  1. Styl lokalny - styl dodawany bezpośrednio do znacznika za pomocą atrybutu style: 
    <span style="...">...</span>
  2. By użyć wewnętrznego arkusza stylów, trzeba to dodać do <head>: 
    <style>
...
</style>
  3. Zewnętrzny arkusz stylów importuje się za pomocą: 
    <link rel="stylesheet" href="...">

    Jest to najszerzej stosowana metoda dodawania arkusza stylów w prawdziwych stronach internetowych (z tego, co wiem).

  4. Import arkusza stylów. Arkusz stylów można włączyć do innego arkusza stylów za pomocą @import url("...");.
  5. Domyślne style przeglądarki. Nie są one zalecane, ponieważ uzależniają wygląd strony od używanej przeglądarki.

Podstawowe selektory

Jest kilka podstawowych rodzajów selektorów, które można używać. Choć tu nie zaprezentuję wszystkich, to są pewne podstawowe, które mogą się okazać przydatne:

  • znacznik - podstawowy selektor, używany do zaaplikowania reguły wszystkim znacznikom określonego typu: 
    p { color: blue; }
<p>Ten tekst będzie niebieski.</p>
<p>Ten tekst też będzie niebieski.</p>
  • #id - pozwala wybrać jeden, określony element: 
    #kolor { color: blue; }
<p>Ten tekst nie będzie niebieski...</p>
<p id="kolor">...ale ten już będzie, i to czyni go wyjątkowym.</p>
  • .class - wybiera zestaw ręcznie określonych elementów, niezależnie od typu znacznika: 
    .kolor { color: blue; }
<p>Ten tekst nie będzie niebieski...</p>
<p class="kolor">...ale ten już będzie.</p>
<div class="kolor">Co zaskakujące, ten też będzie.</div>
  • s1, s2 - pozwala zaaplikować styl elementom wybieranym przez wiele selektorów: 
    #kolor, div { color: blue; }
<p class="kolor">Ten tekst będzie niebieski.</p>
<div>Ten tekst też będzie niebieski.</div>
  • * - selektor uniwersalny, jest aplikowany wszystkim znacznikom.

Zanim zaczniesz

Pamiętaj, by po każdej deklaracji umieszczać średnik. By już nie popełniać takich błędów, możesz użyć Walidatora CSS.

I ostatnia rzecz: w CSS też istnieją komentarze. Będą one ignorowane przez CSS.
